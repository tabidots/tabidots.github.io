Markdown

Install language-gfm-enhanced (supports more languages and \(\LaTeX\))

pip3 install markdown-kernel
python3 -m markdown_kernel.install

Change Language Mappings in Hydrogen settings to: 

{ "python": "magicpython", "markdown": "github markdown"}

This setup highlights delimited \(\LaTeX\) and \(\KaTeX\) properly within Markdown files, but rendering is a little slow (i.e., you might get a blank preview the firs time), and the \(\KaTeX\)-specific syntax is not supported by the renderer.
