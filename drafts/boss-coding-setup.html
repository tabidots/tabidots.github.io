Python

Python 2 comes built-in. It’s better not to mess with the native Python installation too much, since some parts of the system depend on it.

Get Python 3, virtualenv to isolate your projects, git because how else will you keep track of your stuff, and iPython to give yourself a Python shell that doesn’t suck:

$ brew install python3 git
$ sudo pip3 install virtualenv ipython

And then depending on your use case:

# For simple blogging
$ pip3 install pelican
# For web dev
$ pip3 install django flask gunicorn
# For NLP
$ pip3 install spacy textacy
$ pip3 install numpy torch allennlp
# For data science, machine learning, statistics, etc.
$ brew install jupyter numpy scipy matplotlib pandas
# For deep learning
$ pip3 install tensorflow keras torch # & torchvision, maybe

And for a very special use case (Japanese NLP—source):

$ brew install curl xz
$ brew install crf++

$ brew install mecab mecab-ipadic
$ git clone --depth 1 https://github.com/neologd/mecab-ipadic-neologd.git
$ cd mecab-ipadic-neologd
$ ./bin/install-mecab-ipadic-neologd -n -a
$ mecab -d /usr/local/lib/mecab/dic/mecab-ipadic-neologd/
$ brew install cabocha

$ pip install mecab-python3
$ curl -OL https://github.com/taku910/cabocha/archive/master.zip
$ unzip master.zip
$ cd cabocha-master
$ pip install python/
$ git clone https://github.com/kenkov/cabocha
$ pip install cabocha/

Clojure

Install Clojure

Ensure you’ve got Java 8 (1.8) installed set as your $JAVA_HOME, regardless of whether you have other versions of Java installed as well. Leiningen doesn’t work with versions of Java beyond 8!

$ /usr/libexec/java_home -V   # list all installed versions
$ java -version               # show active version
# If Java 8 is not installed
$ brew install java8    
# Or, if Java 8 is not the active version
$ export JAVA_HOME=$(/usr/libexec/java_home -v 1.8)

$ brew install clojure

Pimp your REPL

It’s worth installing this fantastic readline tool to turn your terminal into an interactive version of the Clojure docs (even if you write the bulk of your code in Atom).

It seems that LightTable (which I used to use specifically for its built-in implementation of this) is no longer being maintained—too bad, I liked it quite a lot when it was new.

Append this to your .zshrc (or .bash_profile) file:

# source: https://gist.github.com/Lokeh/a67bb2dc9295d7990d9f8525f478457c
function rebel-clj() {
  clojure -Sdeps "{:deps {com.bhauman/rebel-readline {:mvn/version \"0.1.4\"} $@}}" -m rebel-readline.main
}
function rebel-cljs() {
  clojure -Sdeps "{:deps {com.bhauman/figwheel-main {:mvn/version \"0.1.7\"} com.bhauman/rebel-readline-cljs {:mvn/version \"0.1.4\"} $@}}" -m figwheel.main
}
